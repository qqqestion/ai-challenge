
# День 31: Голосовой агент (Speech → LLM → Text)

## Описание

Нужно соединить распознавание речи и LLM в одну цепочку: **голосовая команда → текст (STT) → запрос в модель → текстовый ответ**. На выходе должен получаться именно текст (без обязательного озвучивания).

## Что нужно сделать

- Подключить модуль распознавания речи (STT) и получить текст из голосового ввода.
- Передавать распознанный текст в LLM (используя текущую интеграцию/клиент модели в проекте).
- Возвращать ответ модели в виде текста (печать в консоль/ответ в чат-боте — любой текстовый канал).
- Протестировать пайплайн на нескольких запросах, например:
  - «посчитай …»
  - «дай определение …»
  - «скажи анекдот»
